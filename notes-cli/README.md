## Команды запуска
- Добавить заметку: `java -cp src com.example.App --cmd=add --text="Текст"`
- Показать список: `java -cp src com.example.App --cmd=list`
- Удалить заметку: `java -cp src com.example.App --cmd=rm --id=1`
- Посчитать количество: `java -cp src com.example.App --cmd=count`
